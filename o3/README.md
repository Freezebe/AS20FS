# AV-EDR Malware Test
Uppgiften gick ut på att testa datorns antivirus och EDR på om de kan känna igen välkända virsusignaturer, skriptet skapar en fil och skriver in en test signatur och väntar en kort stund för att se om den tas bort eller inte.
Starta skriptet och vänta så skriver den ut själv om filen tagits bort eller inte.
